Können wir Hintergrundbilder mit HTML einfügen?

Diese Frage ist einfach. Und die Antwort darauf lautet Nein. Wir können nur mit
einem Stylsheet (einem CSS Dokument) einen Hintergrund einfügen.
Allerdings können wir mit dem Befehl
style type="text/css"
ein Stylesheet in unsere Website erzeugen.
In dieses Stylesheet tippen wir dann folgendes ein:
body {
background-image: url();
background-repeat: no-repeat;
background-size: cover;
}
 Bitte vergessen sie nicht, die URL durch ihre Ziel URL zu ersetzten.
